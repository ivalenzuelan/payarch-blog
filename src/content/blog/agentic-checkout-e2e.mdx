Agentic checkout is not just a buzzword. It requires a robust, technical orchestration layer capable of translating natural language intents into actual iso8583 financial messages on the Visa and Mastercard networks.

In this interactive diagram, we explain exactly what happens under the hood when an Agent Wallet initiates a checkout flow. Notice how the intent moves through the network and settles on the issuer side.

### The Architecture explained

As you can see, the Agent Wallet does not hold actual fiat directly; instead, it maintains an intent ledger that eventually reconciles via traditional `0100` Authorization requests.
